=== Debut ===

[Debut](https://github.com/kwight/debut/) is a solid, sensible, responsive starter or parent theme, based on _s by Automattic. It is translation-ready, and includes a sliding menu for mobile screens.

== Instructions ==

Make sure to create and set a menu under Appearance > Menus. Once you've made your menu, select it under Theme Locations > Primary Menu (this is necessary for the sub-menu indicators to appear).
